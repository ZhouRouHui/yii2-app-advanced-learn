以下是 yii2 advance 版本的一些使用笔记安装和启动    1. composer 安装        composer create-project --prefer-dist yiisoft/yii2-app-advanced advanced    2. 执行 init 文件或 init.bat 文件        - mac 或 linux 系统通过 php 执行 init 文件，也就是 `php init`        - windows 系统直接双击执行 init.bat 文件        命令行会提示配置当前环境，然后生成几个文件，这几个文件默认都是 git 忽略的    3. 创建数据库，并在 common/config/main-local.php 中配置数据库链接    4. 执行 `php yii migrate` 数据迁移，migration 文件在 console/migrations 目录中别名    在 common/config/bootstrap.php 中设置了很多别名，设置好的别名在别的配置文件中可以直接使用gii 的使用，gii 是 yii 框架的脚手架工具，快速生成代码文件    1. 在 backend/config/main-local.php 或 frontend/config/main-local.php 中开启 gii 模块    2. 然后访问前台或者后台对应的页面，注意 r=gii 的参数，例如：        http://localhost/yii2-app-advanced/backend/web/index.php?r=gii    Model Generator 生成模型文件    CRUD Generator 生成控制器，视图等文件
